Elevate CSS features a single compound property for setting the selection behavior of an element:

<br/>

### `select` Property Modifiers

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `select` |  `none`, `auto`, `text`, `contain`, `all` |  `select:auto`  |
