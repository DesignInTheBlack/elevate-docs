## Scroll Snap

| Property | Expected Values | Example |
|----------|------------------|---------|
| `scroll-snap-type` | `none`, `x`, `y`, `block`, `inline`, `both`, `x-mandatory`, `y-mandatory`, `block-mandatory`, `inline-mandatory`, `both-mandatory`, `x-proximity`, `y-proximity`, `block-proximity`, `inline-proximity`, `both-proximity` | `scroll-snap-type:x-mandatory` |
| `scroll-snap-align` | `none`, `start`, `end`, `center` | `scroll-snap-align:center` |
