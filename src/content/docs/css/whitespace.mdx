---
title: Whitespace
description: A guide to whitespace in Elevate CSS.
---

Elevate CSS features a compound property for setting the whitespace behaviour of an element:

<br/>

### The `white-space` Property

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `white-space` | `no-wrap`, `normal`, `pre-line`, `pre-wrap`, `pre`, `break-spaces` |  `white-space:normal`  |
