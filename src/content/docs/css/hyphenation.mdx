Elevate CSS features a compound property for setting the hyphenation behavior of text:

<br/>

### The `hyphenation` Property

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `hyphenation` |  `none`, `auto`, `manual` |  `hyphenation:auto`  |
