Elevate CSS features a compound property for setting the color of a resizing caret:

<br/>

### The `caret` Property

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `caret` |  ColorToken (e.g., `purple-light`) |  `caret:purple-light`  |
