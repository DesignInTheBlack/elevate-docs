Elevate CSS features a single compound property for setting the color of a resizing caret:

<br/>

### `caret` Property Modifiers

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `caret` |  ColorToken (e.g., `purple+1`) |  `caret:purple+1`  |
