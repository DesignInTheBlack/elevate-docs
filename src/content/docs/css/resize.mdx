title: Resize
description: A guide to resize in Elevate CSS.
---

Elevate CSS features a single compound property for setting the resize behavior of an element:

<br/>

### `resize` Property Modifiers

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `resize` |  `none`, `both`, `block`, `inline`, `horizontal`, `vertical` |  `resize:horizontal`  |
