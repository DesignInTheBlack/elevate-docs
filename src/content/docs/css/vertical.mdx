---
title: Vertical Alignment
description: A guide to vertical alignment in Elevate CSS.
---

Elevate CSS features a compound property for setting the vertical alignment of an element:

<br/>

### `vertical` Property Modifiers

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `vertical` |  `baseline`, `sub`, `super`, `top`, `text-top`, `middle`, `bottom`, `text-bottom`  |  `vertical:middle`  |
