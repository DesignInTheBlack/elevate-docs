---
title: Box Sizing
description: A guide to box sizing in Elevate CSS.
---

Elevate CSS features two direct properties for setting box-sizing, plus a compound property for `box-decoration-break`:

<br/>

| Property  |  Example     |
|--------------|---------------------|
| `border-box` |  `border-box`  |
| `content-box` |  `content-box`  |
| `box-decoration-break` |  `box-decoration-break:slice`  |

<br/>

`box-decoration-break` accepts `slice` or `clone`.
