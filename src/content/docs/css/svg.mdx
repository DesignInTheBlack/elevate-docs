## SVG Properties

| Property | Expected Values | Example |
|----------|------------------|---------|
| `fill` | ColorToken (e.g., `purple+1`) | `fill:purple+1` |
| `stroke` | ColorToken (e.g., `indigo+2`) | `stroke:indigo+2` |
| `stroke-width` | SpacingToken (e.g., `d1`) | `stroke-width:d1` |
