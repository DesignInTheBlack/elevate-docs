## Scroll Margin & Padding

| Property | Expected Values | Example |
|----------|------------------|---------|
| `scroll-margin` | SpacingToken | `scroll-margin:d3` |
| `scroll-padding` | SpacingToken | `scroll-padding:d2` |

<br/>

Note: `scroll-margin` and `scroll-padding` currently accept a single value.
