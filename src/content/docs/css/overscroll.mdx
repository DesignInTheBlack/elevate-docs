## Overscroll Behavior

| Property | Expected Values | Example |
|----------|------------------|---------|
| `overscroll` | `auto`, `contain`, `none` | `overscroll:contain` |
| `overscroll-x` | `auto`, `contain`, `none` | `overscroll-x:none` |
| `overscroll-y` | `auto`, `contain`, `none` | `overscroll-y:contain` |
