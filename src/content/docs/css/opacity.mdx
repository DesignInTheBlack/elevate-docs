Elevate CSS features a compound property for setting the opacity of an element:

<br/>

### The `opacity` Property

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `opacity` |  NumericToken (e.g., `0.2`) |  `opacity:0.2`  |
