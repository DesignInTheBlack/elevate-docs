Elevate CSS features a compound property for setting the color of a resizing caret:

<br/>

### The `gradient` Property

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `gradient` |  Gradienttoken (e.g., `fadetoblack`) |  `gradient:fadetoblack`  |

<br/>
<br/>

import { Aside } from '@astrojs/starlight/components';

<Aside title="Effects are Token Driven">
As with all other effects in Elevate CSS, gradients are considered to be a "design decision" and as such are treated as design system tokens. This means you can override the preset options, bring your own, etc
</Aside>
