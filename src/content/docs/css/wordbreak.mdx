Elevate CSS features a compound property for setting the wordbreak behaviour of an element:

<br/>

### The `break` Property

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `break` | `normal`, `all`, `keep`, `word` |  `break:normal`  |
