Elevate CSS features a shared compound property for applying shadows to both elements as well as text:

<br/>

### The `shadow` Property

| Modifier  | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `shadow` |  ShadowToken (e.g., `soft`) |  `shadow:soft`  |


<br/>
<br/>

<Aside title="Effects are Token Driven">
As with all other effects in Elevate CSS, shadows are considered to be a "design decision" and as such are treated as design system tokens. This means you can override the preset options, bring your own, etc
</Aside>
