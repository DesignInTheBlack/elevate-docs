Elevate CSS features compound property for specifying the overflow of an element:

<br/>

### `overflow` Property Modifiers

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `overflow` |  `visible`, `hidden`, `scroll`, `auto`  |  `overflow:hidden`  |
