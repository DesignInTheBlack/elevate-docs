Elevate CSS features a singular compound property with a wide range of modifiers for setting aspect ratios:

<br/>

### `aspect` Property Modifiers

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `aspect` |  `auto` (auto)<br/> `square` (1/1)<br/> `video` (16/9)<br/> `widescreen` (21/9)<br/> `banner` (3/1)<br/> `section` (4/3)<br/> `landscape` (16/10)<br/> `portrait` (3/4)<br/> `pano` (2/1)<br/> `instagram` (4/5)<br/> `thumbnail` (4/3)<br/> `card` (7/5)<br/> `gallery` (1.5/1)<br/> `feature` (5/4)<br/> `fb-post` (1.91/1)<br/> `wide-banner` (5/1)   |  `aspect:video`  |
