title: Aspect Ratios
description: A guide to aspect ratios in Elevate CSS.
---

Elevate CSS features a singular property with a wide range of modifiers for setting aspect ratios:

<br/>

### The `aspect` Property

| Property   | Expected Values | Example     |
|------------|--------------------|-----------------------|
| aspect |  `auto` (auto), `square` (1/1), `video` (16/9), `widescreen` (21/9), `banner` (3/1), `section` (4/3), `landscape` (16/10), `portrait` (3/4), `pano` (2/1), `instagram` (4/5), `thumbnail` (4/3), `card` (7/5), `gallery` (1.5/1), `feature` (5/4), `fb-post` (1.91/1), `wide-banner` (5/1)   |  `aspect:video`  |
