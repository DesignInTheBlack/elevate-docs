Elevate CSS features a compound property for setting the color of a resizing caret:

<br/>

### The `outline` Property

| Modifier  | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `color` |  ColorToken (e.g., `purple-light`) |  `outline:purple-light`  |
| `width` |  SpacingToken (e.g., `d4`) |  `outline:d4`  |
| `style` |  `solid`, `dashed`, `dotted`, `double`, `groove`, `ridge`, `inset`, `outset`, `none`, `hidden`|  `outline:dotted`  |
| `offset` |  SpacingToken (e.g., `d4`) |  `outline:off-d4`  |
| `radius` |  SpacingToken (e.g., `d4`) |  `outline:rad-d4`  |
