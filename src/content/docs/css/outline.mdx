title: Outline
description: A guide to outline in Elevate CSS.
---

Elevate CSS features a compound property for setting outline color, width, style, and offset:

<br/>

### `outline` Property Modifiers

| Modifier  | Expected Values | Example     |
|------------|--------------------|-----------------------|
| `color` |  ColorToken (e.g., `purple+1`) |  `outline:purple+1`  |
| `width` |  SpacingToken (e.g., `d4`) |  `outline:d4`  |
| `style` |  `solid`, `dashed`, `dotted`, `double`, `groove`, `ridge`, `inset`, `outset`, `none`, `hidden`|  `outline:dotted`  |
| `offset` |  SpacingToken (e.g., `d4`) |  `outline:off-d4`  |
| `radius` |  SpacingToken (e.g., `d4`) |  `outline:round-d4`  |

<br/>

Note: `outline:round-*` sets `border-radius` on the element (there is no native `outline-radius` property).
